This is a tiny NAS homeserver that is very lightweight

Quick overview
- List files in a storage directory
- Upload files from a browser
- Serve uploaded files so they can be downloaded

Prerequisites
- Node.js 
- npm
- 
Quick start (5 minutes)
1. Clone this repo
```bash
git clone https://github.com/XaydenY/homelab.git
cd homelab
```

2. Install dependencies
```bash
npm install
```


3. Start the dev server
```bash
npm run dev
```

4. Open your browser at http://localhost:3000 and try viewing a file.
