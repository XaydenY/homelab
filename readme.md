# Homelab Project

## Structure
- `backend/`: Node.js/Express backend server
- `frontend/`: React frontend app

## Getting Started

### Backend
1. `cd backend`
2. `npm install`
3. `npm start`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm start`

---

## Description
This project is a template for a homelab server with a Node.js backend and a React frontend. Each can be developed and run independently.
